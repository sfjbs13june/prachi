## How to create Spring boot application

https://start.spring.io/


## Build application
``` 
mvn clean install
```

## Run application
``` 
mvn spring-boot:run
```

## Test Application
1) Unit Test - Calling the methods
2) Integration Test - Calling the Controller Class
``` 
mvn clean test
```

## Code Coverage

Right Click on project --> Run 'All Tests' with coverage
